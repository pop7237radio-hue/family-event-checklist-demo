# 家族の予定・持ち物チェック 公開用パッケージ

このフォルダは、GitHub Pages で公開するための最小セットです。

個人メモや制作途中の管理ファイルは含めず、公開してよい家族予定・持ち物管理デモだけを入れています。

## 中身

- `index.html`
  - 家族の予定・持ち物チェック
  - 配布資料スクショ添付
  - スクショ文字読み取り
  - 読み取った日付・予定名・持ち物の下書き自動反映
  - 子どもの名前登録
  - 保育園・小学校・仕事別の持ち物テンプレ
  - その他持ち物の手入力追加

## GitHub Pages 公開後のURL

- `https://pop7237radio-hue.github.io/family-event-checklist-demo/`

## 注意

- 入力データは各ユーザーのブラウザ内に保存されます。
- 本番納品前のデモ版です。
- スクショ文字読み取りと下書き反映はブラウザ内OCRのため、画像の明るさや文字の小ささで精度が変わります。
- 家族共有、通知、バックアップは未対応です。
